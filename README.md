## Getting Started

### Prerequisites

1. LTS version of [Node](https://nodejs.org/).
2. Yarn
  ```sh
  npm install -g yarn
  ```

### Installation

1. Clone the repo
  ```sh
  git clone https://github.com/carles2m/carles2m.com.git
  ```
2. Install dependencies
  ```sh
  cd carles2m.com
  yarn
  ```

### Deployment

* Development build
  ```sh
  yarn dev
  ```
* Production build
  ```sh
  yarn build
  yarn start
  ```
